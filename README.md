# 個人財務報表

簡潔、單檔、純前端的個人財務工具。月度損益表 + 資產負債表，支援多月份切換、列印 PDF、匯出美化過的 Excel。

> **資料隱私**：所有輸入只儲存在你自己的瀏覽器（localStorage），不會上傳任何伺服器，也不會跨裝置同步。

## 功能

- **損益表**：收入 / 支出，自動算總收入、總支出、月現金流（正綠負紅）
- **資產負債表**：資產 / 負債，自動算總資產、總負債、淨值
- **完全自訂項目**：每個項目名稱可以自己改，可以新增、刪除、拖曳排序
- **多月份分頁**：頂部分頁切換，新增月份會自動套用上月的項目結構（金額留空）
- **自動暫存**：每次輸入即存入瀏覽器 localStorage
- **匯出 Excel**：使用 `xlsx-js-style`，含「月份彙總」橫向比對 + 每月一個分頁，含逐月變化差額
- **匯出 / 載入 JSON**：跨裝置 / 跨瀏覽器搬資料用
- **列印 / PDF**：呼叫瀏覽器列印對話框，A4 排版好

## 使用方式

1. 直接開 [https://你的帳號.github.io/你的repo名/](#) 即可使用
2. 或下載 `index.html` 雙擊用 Chrome / Edge 打開

無需註冊、無需安裝、無需後端。

## 技術

- 純 HTML / CSS / Vanilla JavaScript，單一檔案
- 唯一外部相依：[`xlsx-js-style`](https://github.com/gitbrent/xlsx-js-style)（CDN）做 Excel 樣式匯出
- 拖曳排序使用原生 HTML5 Drag and Drop API

## 部署到 GitHub Pages

1. 將整個資料夾推上 GitHub repo
2. Repo Settings → Pages → Source 選 `main` 分支的根目錄
3. 等幾分鐘，網址會出現在 Pages 設定頁

## 備份建議

localStorage 在以下情況會消失：清除瀏覽器資料、無痕模式、換瀏覽器、換裝置。建議：

- 每次填完按「下載 JSON」存一份檔案在電腦或雲端硬碟
- 月底時按「匯出 Excel」做永久存檔
