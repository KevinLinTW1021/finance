👋 個人財務報表 · 使用說明

這是一個簡潔的個人月度財務工具。所有資料只存在你目前這個瀏覽器，不會上傳任何伺服器。

基本使用

  • 右上方「姓名」填上你的名字（一次就好）
  • 「損益表」記錄這個月的收入與支出，自動算「月現金流」
  • 「資產負債表」記錄目前的資產與負債餘額，自動算「淨值」
  • 每打一個字就自動儲存，關掉再打開資料還在

編輯每一列

  • 項目名稱可以直接改：點左邊文字（例如把「工資」改成「主業薪資」）
  • 調整順序：按住左側的 ⋮⋮ 把手往上下拖
  • 刪除：點右邊的 ×
  • 新增：點分區下方虛線框的「＋ 新增…」
  • 股票/基金：「資產」區下方有「持股明細」子區，填名稱 / 股數 / 每股成本 → 自動算合計並加進總資產（適合看平均成本）
  • 備註區：持股明細與負債之間有個大文字框，可以記任何想記的事（例如「這個月有兼職」「下個月會買 0050」「記得繳房屋稅 8000」）。每個月各自獨立

每月怎麼用

  • 頂部的「月份：2026-05」是分頁，點選切換
  • 下個月填表時點「＋ 新增月份」會自動建立新分頁，內容繼承上個月（你只要修有變動的部分）
  • 分頁雙擊可改名（例如「2026-05」→「5月-換工作」）
  • 分頁右上的 × 刪除整個月份（會二次確認）

年度費用註記

  • 右上「📌 年度費用」按鈕可記錄每年固定要繳的費用（車險、稅金、年費⋯⋯），填名稱 / 月份 / 金額
  • 切到對應月份時，主頁會出現黃色提醒條，讓你別忘了把這筆加到當月支出
  • 金額不會自動加進每月支出，只是純提醒清單
  • 匯出 Excel 會多出一個「年度費用」分頁列出所有項目與年度總計
⚠️ 重要：定期備份你的資料
你的資料只存在這個瀏覽器。清除瀏覽器資料、換瀏覽器、換裝置、換電腦都會看不到。

建議的備份習慣：
  1. 每月填完按右上「下載 JSON」
  2. 把下載的檔案存到你的雲端硬碟（Google Drive / Dropbox / iCloud Drive 等）
  3. 檔名會自動加上當天日期，不會覆蓋舊備份

換到另一台電腦 / 瀏覽器

  1. 新裝置打開同一個網址
  2. 右上按「載入 JSON」
  3. 選擇你之前備份的最新 JSON 檔
  4. 所有月份與資料都會還原

三種匯出，用途不同

  • 匯出 Excel：產生美化過的財務報表，含「月份彙總」橫向比對 + 每月各一張分頁。給家人看、給會計師、自己歸檔都好用。
  • 列印 / 存成 PDF：在 Chrome 列印對話框選「另存為 PDF」，印出目前看的月份（A4 排版）。
  • 下載 JSON：跨裝置搬資料 / 備份還原用，內容是純文字，可以還原到頁面。

關於隱私

本頁面不會把任何資料傳給任何伺服器，包含製作者也看不到你的資料。下載 JSON 後上傳到雲端硬碟時，雲端業者（Google / Dropbox 等）技術上能看到內容；如果非常在意，上傳前可以用 7-Zip 之類的工具把 JSON 加上壓縮密碼再傳。

常見問題

  • 不小心刪錯月份了？載入最近一份 JSON 備份就能還原
  • 看到的不是最新資料？檢查是不是在不同瀏覽器或無痕模式打開
  • 金額顯示「–」？那是 0 的顯示樣式（千分位格式），表示沒填
